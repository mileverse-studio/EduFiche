1 Nombres et Arithmétique

Ensembles fondamentaux

  • $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$ (entiers naturels)
  • $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ (entiers relatifs)

Divisibilité dans $\mathbb{Z}$

📖 Définitions

$a$ divise $b$ (notation $a | b$) signifie $\exists k \in \mathbb{Z}$ tel que $b = ka$

PGCD et PPCM

Propriété Formule
Relation PGCD-PPCM $\text{PGCD}(a,b) \times \text{PPCM}(a,b) = |a \times b|$
Algorithme d'Euclide $\text{PGCD}(a,b) = \text{PGCD}(b,r)$ où $a = bq + r$

Nombres premiers

💡 Définition

Un nombre premier $p$ est un entier $> 1$ dont les seuls diviseurs sont $1$ et $p$

  • Tout entier $n > 1$ se décompose en produit de facteurs premiers
  • $n = p_1^{\alpha_1} \times p_2^{\alpha_2} \times \cdots \times p_k^{\alpha_k}$
  • Cette décomposition est unique à l'ordre des facteurs près

2 Barycentre et Produit Vectoriel

Barycentre de n points pondérés

📖 Définition

Le barycentre $G$ de $(A_1,\alpha_1), (A_2,\alpha_2), \ldots, (A_n,\alpha_n)$ avec $\sum \alpha_i \neq 0$ vérifie :

$$\alpha_1 \overrightarrow{GA_1} + \alpha_2 \overrightarrow{GA_2} + \cdots + \alpha_n \overrightarrow{GA_n} = \overrightarrow{0}$$

Coordonnées du barycentre

Si $G(x_G, y_G)$ et $A_i(x_i, y_i)$, alors :

$$x_G = \frac{\sum_{i=1}^{n} \alpha_i x_i}{\sum_{i=1}^{n} \alpha_i}, \quad y_G = \frac{\sum_{i=1}^{n} \alpha_i y_i}{\sum_{i=1}^{n} \alpha_i}$$

Produit vectoriel

📖 Définition

Le produit vectoriel $\overrightarrow{w} = \overrightarrow{u} \wedge \overrightarrow{v}$ vérifie :

  • $\overrightarrow{w} \perp \overrightarrow{u}$ et $\overrightarrow{w} \perp \overrightarrow{v}$
  • $(\overrightarrow{u}, \overrightarrow{v}, \overrightarrow{w})$ forme un repère direct
  • $\|\overrightarrow{w}\| = \|\overrightarrow{u}\| \cdot \|\overrightarrow{v}\| \cdot |\sin(\overrightarrow{u}, \overrightarrow{v})|$
PropriétéFormule
Antisymétrie$\overrightarrow{u} \wedge \overrightarrow{v} = -\overrightarrow{v} \wedge \overrightarrow{u}$
Linéarité$(\lambda \overrightarrow{u} + \mu \overrightarrow{v}) \wedge \overrightarrow{w} = \lambda \overrightarrow{u} \wedge \overrightarrow{w} + \mu \overrightarrow{v} \wedge \overrightarrow{w}$
Double produit$\overrightarrow{u} \wedge (\overrightarrow{v} \wedge \overrightarrow{w}) = \overrightarrow{v}(\overrightarrow{u} \cdot \overrightarrow{w}) - \overrightarrow{w}(\overrightarrow{u} \cdot \overrightarrow{v})$
Produit mixte$(\overrightarrow{u} \wedge \overrightarrow{v}) \cdot \overrightarrow{w} = \overrightarrow{u} \cdot (\overrightarrow{v} \wedge \overrightarrow{w})$

Applications du produit vectoriel

  • Aire du parallélogramme : $\mathcal{A} = \|\overrightarrow{AB} \wedge \overrightarrow{AC}\|$
  • Aire du triangle : $\mathcal{A} = \frac{1}{2} \|\overrightarrow{AB} \wedge \overrightarrow{AC}\|$
  • Volume du parallélépipède : $\mathcal{V} = |(\overrightarrow{AB} \wedge \overrightarrow{AC}) \cdot \overrightarrow{AD}|$
  • Volume du tétraèdre : $\mathcal{V} = \frac{1}{6} |(\overrightarrow{AB} \wedge \overrightarrow{AC}) \cdot \overrightarrow{AD}|$

3 Nombres Complexes

Forme algébrique

📖 Définition

Tout nombre complexe $z$ s'écrit de manière unique :

$$z = a + bi \quad \text{avec } a, b \in \mathbb{R} \text{ et } i^2 = -1$$

$a = \text{Re}(z)$ (partie réelle), $b = \text{Im}(z)$ (partie imaginaire)

OpérationFormule
Conjugué$\overline{z} = a - bi$
Module$|z| = \sqrt{a^2 + b^2}$
Inverse$\dfrac{1}{z} = \dfrac{\overline{z}}{|z|^2}$ pour $z \neq 0$
Quotient$\dfrac{z_1}{z_2} = \dfrac{z_1 \overline{z_2}}{|z_2|^2}$ pour $z_2 \neq 0$
Multiplication$(a+bi)(c+di) = (ac-bd) + (ad+bc)i$

Forme trigonométrique

💡 Formule

$$z = r(\cos \theta + i \sin \theta) = re^{i\theta}$$

où $r = |z|$ et $\theta = \arg(z)$

Propriétés des arguments

  • $\arg(z_1 z_2) = \arg(z_1) + \arg(z_2)$
  • $\arg\left(\dfrac{z_1}{z_2}\right) = \arg(z_1) - \arg(z_2)$
  • $\arg(z^n) = n \arg(z)$
📖 Formule de Moivre

$$(\cos \theta + i \sin \theta)^n = \cos(n\theta) + i \sin(n\theta)$$

📖 Formules d'Euler

$$\cos \theta = \frac{e^{i\theta} + e^{-i\theta}}{2} \quad ; \quad \sin \theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}$$

Racines n-ièmes

💡 Racines de l'unité

$$z_k = e^{2ik\pi/n} = \cos\left(\frac{2k\pi}{n}\right) + i\sin\left(\frac{2k\pi}{n}\right), \quad k = 0, 1, \ldots, n-1$$

Si $z = re^{i\theta}$, les $n$ racines $n$-ièmes sont :

$$z_k = r^{1/n} e^{i(\theta + 2k\pi)/n}, \quad k = 0, 1, \ldots, n-1$$

Applications affines

📖 Expression analytique

Dans un repère $(O, \vec{i}, \vec{j})$, une application affine s'écrit :

$$\begin{cases} x' = ax + by + c \\ y' = a'x + b'y + c' \end{cases}$$

Propriétés : l'image d'une droite est une droite, le parallélisme est conservé.

Similitudes directes

💡 Définition et écriture complexe

Une similitude directe a pour écriture complexe :

$$z' = az + b \quad \text{avec } a \in \mathbb{C}^*$$

  • Rapport : $k = |a|$
  • Angle : $\theta = \arg(a)$
  • Centre : solution de $z = az + b$

5 Applications de l'Espace

Projections

La projection sur un plan $\Pi$ parallèlement à une droite $\Delta$ vérifie :

  • $M' \in \Pi$ et $(MM') \parallel \Delta$
  • Les points de $\Pi$ sont invariants

  • Symétrie centrale : $\overrightarrow{OM'} = -\overrightarrow{OM}$
  • Demi-tour : composée de deux symétries planes perpendiculaires à l'axe

6 Coniques

📖 Définition générale

Une conique $\mathcal{C}$ est l'ensemble des points $M$ tels que :

$$\frac{MF}{d(M, \Delta)} = e \quad \text{(excentricité)}$$

  • Si $e < 1$ : ellipse
  • Si $e = 1$ : parabole
  • Si $e > 1$ : hyperbole

Parabole

$$y^2 = 2px \quad (p > 0)$$

  • Foyer : $F\left(\frac{p}{2}, 0\right)$
  • Directrice : $x = -\frac{p}{2}$
  • Excentricité : $e = 1$

Ellipse

💡 Équation réduite

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \quad \text{avec } a \geq b > 0$$

  • Sommets : $(\pm a, 0)$ et $(0, \pm b)$
  • Foyers : $F(c, 0)$ et $F'(-c, 0)$ avec $c^2 = a^2 - b^2$
  • Excentricité : $e = \frac{c}{a} < 1$
  • Propriété focal : $MF + MF' = 2a$

Hyperbole

💡 Équation réduite

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \quad \text{avec } a, b > 0$$

  • Sommets : $(\pm a, 0)$
  • Foyers : $F(c, 0)$ et $F'(-c, 0)$ avec $c^2 = a^2 + b^2$
  • Excentricité : $e = \frac{c}{a} > 1$
  • Asymptotes : $y = \pm \frac{b}{a}x$
  • Propriété focal : $|MF - MF'| = 2a$

7 Limites et Continuité

Limites usuelles

Fonction$\lim_{x \to +\infty}$$\lim_{x \to -\infty}$
$x^n$ (n pair)$+\infty$$+\infty$
$x^n$ (n impair)$+\infty$$-\infty$
$\dfrac{1}{x^n}$$0$$0$
$\sqrt{x}$$+\infty$N'existe pas

Opérations sur les limites

Si $\lim f = L_1$ et $\lim g = L_2$ :

  • Somme : $\lim(f+g) = L_1 + L_2$
  • Produit : $\lim(f \cdot g) = L_1 \cdot L_2$
  • Quotient : $\lim\frac{f}{g} = \frac{L_1}{L_2}$ si $L_2 \neq 0$
⚠️ Formes indéterminées

$$\infty - \infty \quad ; \quad 0 \times \infty \quad ; \quad \frac{\infty}{\infty} \quad ; \quad \frac{0}{0} \quad ; \quad 1^\infty \quad ; \quad 0^0 \quad ; \quad \infty^0$$

Continuité

📖 Définition

$f$ est continue en $a$ si $\lim_{x \to a} f(x) = f(a)$

📖 Théorème des Valeurs Intermédiaires (TVI)

Si $f$ est continue sur $[a,b]$ et si $f(a) \cdot f(b) < 0$, alors $\exists c \in ]a,b[$ tel que $f(c) = 0$

Asymptotes

  • Verticale : $x = a$ si $\lim_{x \to a} f(x) = \pm\infty$
  • Horizontale : $y = b$ si $\lim_{x \to \pm\infty} f(x) = b$
  • Oblique : $y = mx + p$ si $\lim_{x \to \pm\infty} [f(x) - (mx + p)] = 0$

avec $m = \lim \frac{f(x)}{x}$ et $p = \lim [f(x) - mx]$

📖 Théorème des Accroissements Finis (TAF)

Si $f$ est continue sur $[a,b]$ et dérivable sur $]a,b[$, alors $\exists c \in ]a,b[$ tel que :

$$f(b) - f(a) = f'(c)(b - a)$$

Tangente à une courbe

Équation de la tangente en $x = a$ :

$$y = f'(a)(x - a) + f(a)$$

📖 Définition

12 Intégration

Primitives usuelles

📖 Théorème fondamental

$$\int_a^b f(x) \, dx = F(b) - F(a)$$

où $F$ est une primitive de $f$ sur $[a,b]$

Propriétés de l'intégrale

  • Linéarité : $\int (af + bg) = a\int f + b\int g$
  • Relation de Chasles : $\int_a^c f = \int_a^b f + \int_b^c f$
  • Positivité : Si $f \geq 0$ sur $[a,b]$, alors $\int_a^b f \geq 0$
💡 Intégration par parties

$$\int u \, dv = uv - \int v \, du$$

Applications

  • Aire : $A = \int_a^b |f(x)| \, dx$
  • Valeur moyenne : $\mu = \dfrac{1}{b-a} \int_a^b f(x) \, dx$

$$y = Ce^{ax} \quad \text{où } C \in \mathbb{R}$$

$$y = y_0 e^{a(x-x_0)}$$

⭐ FORMULES ESSENTIELLES

Dérivées fondamentales

  1. $(x^n)' = nx^{n-1}$
  2. $(\sin x)' = \cos x$
  3. $(\cos x)' = -\sin x$
  4. $(e^x)' = e^x$
  5. $(\ln x)' = \dfrac{1}{x}$

Primitives fondamentales

  1. $\int x^n \, dx = \dfrac{x^{n+1}}{n+1} + C$ ($n \neq -1$)
  2. $\int \dfrac{1}{x} \, dx = \ln|x| + C$
  3. $\int e^x \, dx = e^x + C$
  4. $\int \sin x \, dx = -\cos x + C$
  5. $\int \cos x \, dx = \sin x + C$

Limites fondamentales

  1. $\lim_{x \to 0} \dfrac{\sin x}{x} = 1$
  2. $\lim_{x \to 0} \dfrac{e^x - 1}{x} = 1$
  3. $\lim_{x \to 0} \dfrac{\ln(1+x)}{x} = 1$
  4. $\lim_{n \to \infty} \left(1 + \dfrac{1}{n}\right)^n = e$

Nombres Complexes

  1. $z = a + bi = r(\cos \theta + i \sin \theta)$
  2. $(\cos \theta + i \sin \theta)^n = \cos(n\theta) + i \sin(n\theta)$
  3. $\cos \theta = \dfrac{e^{i\theta} + e^{-i\theta}}{2}$, $\sin \theta = \dfrac{e^{i\theta} - e^{-i\theta}}{2i}$

Coniques

  1. Parabole : $y^2 = 2px$, $e = 1$
  2. Ellipse : $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$, $e < 1$, $MF + MF' = 2a$
  3. Hyperbole : $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$, $e > 1$, $|MF - MF'| = 2a$

Trigonométrie

  1. $\sin^2 x + \cos^2 x = 1$
  2. $\sin(a + b) = \sin a \cos b + \cos a \sin b$
  3. $\cos(a + b) = \cos a \cos b - \sin a \sin b$
  4. $\cos(2a) = 2\cos^2 a - 1$
